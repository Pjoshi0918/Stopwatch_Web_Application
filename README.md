 Stopwatch Web Application

A simple and responsive **Stopwatch Web Application** built using HTML, CSS, and JavaScript as **Task 02** of the **Prodigy InfoTech Web Development Internship**.

## 📌 Features

- ▶️ Start Stopwatch
- ⏸️ Pause Stopwatch
- 🔄 Reset Stopwatch
- 📝 Record Lap Times
- 📱 Responsive User Interface
- ⚡ Real-Time Time Updates
- 🎨 Modern and User-Friendly Design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
